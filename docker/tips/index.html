Tips para Docker

1.- Docker stats este comando nos permite ver las estadisticas por contenedor

2.- docker system prune Este comando borra todo lo que no estemos usando (networks imagenes contenedores…)

3 docker inspect +containerid

4 docker cp containerid:/tmp

5 docker-compose restart always on-failure:10

6 docker logs –tail=10 -t containerid

7 docker run –it –entrypoint=bash imagen.
